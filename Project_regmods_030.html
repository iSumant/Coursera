Executive Summary

In this project we work for Motor Trend, a magazine about the automobile industry. Looking at a data set of a collection of cars, we explor the relationship between a set of variables and miles per gallon (MPG) (outcome). We are particularly interested in the following two questions:

  • Is an automatic or manual transmission better for MPG?
  • Quantify the MPG difference between ‘automatic’ and ‘manual’ transmissions.

Solution

1. Data Dictionary

This dataset was extracted from the 1974 Motor Trend US magazine which comprises of 11 aspects of automobile design and performance for 32 automobiles. Refer Table 1(Appendix - A) for the dataset.

data(mtcars)
head(mtcars)

2. Data Transformation

The first step to build any data model is to cleanse the data. As part of data cleansing, we transformed and factored the following categorical variables.

data(mtcars)
mtcars$cyl <- factor(mtcars$cyl)
mtcars$vs <- factor(mtcars$vs)
mtcars$gear <- factor(mtcars$gear)
mtcars$carb <- factor(mtcars$carb)
mtcars$am <- factor(mtcars$am,labels=c('Automatic','Manual'))

3. Data Exploration

Once data transformation was complete, we used multiple data exploration tools such as scatterplots, histograms etc. to identifying meaningful relationships among variables.

scatterPlotMatrix(mtcars)

From the Plot 1(Appendix - B) we notice that variables like cyl, disp, hp, drat, wt, vs and am seem to have some strong correlation with mpg. We also plot a boxplot of the variable mpg grouped by am. Plot 2(Appendix - B) shows that the mpg is higher when the transmission is ‘Manual’.

library(ggplot2)
p <- ggplot(mtcars, aes(am, mpg))
p + geom_boxplot(aes(fill=am))

4. Data Modeling

Let’s investigate the relationship between miles per gallon and transmission type. First we run a regression model with mpg as the response variable and am as the (only) explanatory variable.

mod1<-lm(mpg ~ am, data = mtcars)
summary(mod1)$coef

Table 2(Appendix - A) shows that the model explains only 35% of the total variation in mpg and there are additional variables in our data; there is a chance that we could find a better model. So we run the regression model multiple times using STEP method to build multiple regression models using both forward selection and backward elimination methods by the AIC algorithm.

mod2<-lm(mpg ~ ., data = mtcars)
mod3<-step(mod2, direction = "both")
summary(mod3)

We can see in Table 3(Appendix - A) the adjusted R2 value is equal to 0.84, thus this model can explain 84% of the variability with cyl, wt, hp and am as relevant variables.
We further compare the models with ANOVA. Based on the significant p-values of model coefficients, the model (mpg ~ cyl + hp + wt + am) is our final selection - refer Table 4(Appendix - A).

anova(mod1, mod3)

5. Residuals

par(mfrow = c(2,2)) plot(mod3) Plot 3(Appendix - B) depicts that the residual plots are randomly scattered.

6. Conclusion

From the mod3 (mpg ~ cyl + hp + wt + am) model we can conclude that ‘Manual’ transmission is better than ‘Automatic’ transmission for mpg. Miles per gallon mpg will increase by 1.81 in cars with ‘Manual’ transmission in comparison to cars with ‘Automatic’ transmission (adjusted by cyl, hp & wt).
